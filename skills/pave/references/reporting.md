# Reporting

## Purpose

Leave concise evidence for completed or blocked work.

## Final Report

Include:

- User-visible outcome.
- Files changed.
- Commands run and results.
- Plan path.
- Subagent outcomes when used.
- Memory capture or skip reason.

## Blocked Report

Include:

- Current state.
- Exact blocker.
- Evidence gathered.
- Attempts made.
- Required user input or external change.
